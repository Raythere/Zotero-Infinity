# Changelog

All notable changes to Zotero Infinity will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.1.0] - 2025-02

### Added

- AI chat with papers using local Ollama (llama3.2:1b)
- Tabbed chat sessions per document
- Paper comparison (select multiple papers)
- Library search (title, authors, abstract, full text)
- Smart tagging with AI suggestions
- Library categorization
- Export chat as Zotero note
- One-click Windows installer (bundles Ollama + plugin)
- Auto-retry connection when Ollama starts

### Requirements

- Zotero 7+
- Ollama with llama3.2:1b (or similar)
- ~1 GB disk for model, 4 GB+ RAM recommended
